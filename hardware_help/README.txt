motor_config.edc 

	This file contains all the configuration settings for the maxon motor controller. In the event that the controller must be reprogrammed, this file can be opened in Maxon's Escon Studio Software which can be downloaded here: http://www.maxonmotorusa.com/medias/sys_master/root/8818476810270/ESCON-Setup-7.zip  (A WINDOWS MACHINE IS REQUIRED)

	The RIT Imaging Team provides two preprogrammed motor controllers.
